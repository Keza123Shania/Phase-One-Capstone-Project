# GIT SETUP COMMANDS FOR IGIREPAY

Run these commands in PowerShell in your project directory.
Copy and paste each section sequentially.

## STEP 1: Navigate to project directory
cd c:\Users\Lenovo\IdeaProjects\igirepay-wallet

## STEP 2: Check current git status
git status

## STEP 3: Configure git user (one-time, if not already done)
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Example:
# git config --global user.name "Keza Shania"
# git config --global user.email "keza.shania@example.com"

## STEP 4: Initialize git repository (if not already done)
git init

## STEP 5: Add all files
git add .

## STEP 6: Initial commit
git commit -m "Initial commit: Lab 1 OOP Design and Lab 2 JDBC/DAO implementation

- Lab 1: Account hierarchy (WalletAccount, SavingsAccount)
- Lab 1: Polymorphic transaction processing
- Lab 1: Idempotency detection with HashSet
- Lab 2: PostgreSQL database schema with 4 tables
- Lab 2: DAO pattern (CustomerDAO, AccountDAO, TransactionDAO, ProcessedRequestDAO)
- Lab 2: PreparedStatements for SQL injection prevention
- Lab 2: Duplicate transaction detection at database level"

## STEP 7: Add remote repository
git remote add origin https://github.com/Keza123Shania/Phase-One-Capstone-Project.git

## STEP 8: Verify remote
git remote -v

## STEP 9: Pull any existing content from remote (should be empty or has README)
git pull origin main --allow-unrelated-histories

## STEP 10: Push to main
git push -u origin main

## STEP 11: Create and checkout feature branch for Lab 3
git checkout -b feature/lab3-integration

## STEP 12: Verify you're on the new branch
git branch -a

## STEP 13: Push feature branch to remote
git push -u origin feature/lab3-integration
